== Description ==
Live Order Dashboard is a WordPress plugin designed to integrate with WooCommerce. It provides a live-updating dashboard to view and manage orders in real-time.

== Features ==
- Displays WooCommerce orders in a custom dashboard.
- Real-time updates using AJAX.
- Easy to install and use.

== Installation ==
1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the 'Live Order Dashboard' page in the admin menu.

== Frequently Asked Questions ==
= Does this plugin require WooCommerce? =
Yes, this plugin is designed to work exclusively with WooCommerce.

= How often does the dashboard update? =
By default, the dashboard updates every 5 seconds.

= Can I customize the dashboard? =
You can modify the plugin's CSS or extend its functionality as needed.
